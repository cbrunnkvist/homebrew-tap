# Cbrunnkvist Tap

## How do I install these formulae?

`brew install cbrunnkvist/tap/<formula>`

Or `brew tap cbrunnkvist/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "cbrunnkvist/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
